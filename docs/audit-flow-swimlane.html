Create Audit Request

MODEL WorkflowMilestoneInstance: AR_CREATED

Send Intimation Letter
Open artifact panel, review letter, click "Send to Supplier".

📥 Supplier can now see the audit
System Actions

STATE supplierVisible = true
STATE trackStatus = "Audit intimation sent"
STATE nextAuditOn = "supplier"

MODEL Milestone: INTIMATION_LETTER_SENT
