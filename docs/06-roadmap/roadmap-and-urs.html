HawkEye is a multi-tenant, AI-native supply chain audit and quality management platform with 15 configurable modules, 154 data models, and 78 API endpoints. This document assesses current maturity, maps competitive positioning, identifies gaps, and defines a phased roadmap with detailed user requirements.

7.3 Industry Profiles (9 presets)

Each profile pre-configures modules, vocabulary, workflow keys, and compliance standards:

PHARMA_GMP — Full pharma audit + EQMS. ICH Q7, 21 CFR 211.
MEDICAL_DEVICE — ISO 13485, 21 CFR 820. Design controls needed.
FOOD_SAFETY — FSSC 22000, BRC, SQF. EHS focus.
ISO9001 — Generic quality management. Chemical, manufacturing.
ORGANIC_FARMING — Organic certification audits.
FOREST_COC — Chain of custody for forestry (FSC/PEFC).
REAL_ESTATE — Property inspection workflows.
HIGH_TICKET — High-value asset verification.
CUSTOM — Blank slate — customer configures everything.

7.4 Deployment per Customer

To onboard a new customer, the only configuration needed is a single API call:

PUT /api/universal/module-config
{
  "industryProfile": "PHARMA_GMP",
  "modules": {
    "AUDIT_MANAGEMENT": {"enabled": true},
    "DOCUMENT_CONTROL": {"enabled": true},
    "CHANGE_CONTROL": {"enabled": true},
    ... (toggle per customer)
  },
  "vocabularyOverrides": {
    "audit": "GMP Audit",
    "finding": "Deficiency"
  }
}
